## 💡 What is BloomBot?
BloomBot is an AI-powered adaptive learning app that transforms 
natural human behaviors into a structured cognitive journey 
aligned with Bloom's Taxonomy.

---

## 📚 The 4 Modules

| Module | Bloom's Level | Inspired By |
|--------|--------------|-------------|
| 🔹 Module 1 | Understand & Apply | Doomscrolling behavior |
| 🔹 Module 2 | Analyze & Evaluate | Human desire to correct others |
| 🔹 Module 3 | Recall | The Protégé Effect |
| 🔹 Module 4 | Create & Visualize | Drawing as cognitive relief |

---

## ⚙️ Key Features
- Swipe-based micro-learning cards
- AI student that makes mistakes for users to correct
- Voice-based teach-back interactions
- Digital drawing board for visual reconstruction
